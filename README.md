**# NIFTY 50 Stock Analysis (EDA)

## Overview
This project performs exploratory data analysis on NIFTY 50 stocks to understand market trends, volatility, and relationships between different companies.

## Objectives
- Analyze historical stock data of NIFTY 50 companies
- Identify trends and patterns in stock prices
- Compare volatility across different stocks
- Examine correlations between stock movements

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and preprocessing
- Time series analysis of stock prices
- Visualization of trends and distributions
- Correlation analysis between stocks

## Project File
- `nifty50_eda.ipynb` – complete analysis

## Conclusion
The analysis provides insights into stock behavior, helping understand market patterns and relationships within the NIFTY 50 index.**
